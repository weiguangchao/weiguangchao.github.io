本地电脑配置GitHub SSH key

使用场景：每次在本地操作GitHub远程代码每次都需要手动输入用户名、密码未免太麻烦了。我们将ssh密钥配置到GitHub之后，每次git push都像操作本地一样，不需要在重新输入账号密码。

具体原理类似于ssh远程登录，在远程服务器上配置我们本地的公钥，我们本地通过私钥就可以登录远程服务器了。

1. 获取本地SSH key

1.1. 检查本地是否已经存在rsa密钥

若存在一下两个文件，则代表本地已经存在rsa公私钥。其中，id_rsa是私钥，id_rsa.pub是公钥。我们需要将id_rsa.pub添加到GitHub的密钥对里。若不存在的话，则我们手动创建rsa密钥对。

2. 在GitHub进行添加

2.1. 进入「Settings」选项

点击用户头像，在点击「Settings」选项

2.2. 进入「SSH and GPG keys」选项

点击右侧「SSH and GPG keys」选项

2.3. 添加SSH key

在左侧页面中，点击上方「New SSH key」按钮
